## Why Cardano needs its own MCP

Cardano's eUTxO model is fundamentally different from Ethereum's account model.
There are no balances — only UTxOs. Smart contracts are validators, not code runners.
Datums carry state, not contracts. Amounts are always in lovelace.

Generic blockchain MCPs get this wrong. Cardano MCP is purpose-built around the
correct mental model, so Claude never confuses addresses for accounts or ADA for wei.
