# Installation

## Prerequisites

- **Node.js 20+** — check with `node --version`
- **pnpm** — `npm install -g pnpm`
- A **Blockfrost project ID** — free at [blockfrost.io](https://blockfrost.io)

## Clone and build

```bash
git clone https://github.com/your-org/cardano-mcp
cd cardano-mcp
pnpm install
pnpm build
```

The build step compiles TypeScript and copies resource files to `dist/`.

## Environment variables

Copy the example env file and fill in your keys:

```bash
cp .env.example .env
```

## Verify the build

```bash
node dist/index.js
# Cardano MCP server running (mainnet) via stdio
```

The server speaks MCP over stdio — no port is opened.
