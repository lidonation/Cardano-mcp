Usage rules
DO · clear-space

Reserve 1× root-radius around the mark.

Padding = the radius of the filled root dot. At 64u that's 5u of breathing room on every side. Anchor lockups against this same gutter.

DON'T · alter motion

Never change the flow direction.

Data flows root → leaves. Reversing the dash animation breaks the eUTxO metaphor. If you can't render motion, drop in the static fallback — never mix the two.

DO · solo color

Brand blue for the root and flow lines.

Leaves stay neutral (currentColor) so the mark inherits foreground in any context. Only the root dot + the three flow paths carry the accent.

DON'T · place on busy ground

The flow lines need a calm surface.

Avoid photographs or high-contrast textures behind the mark. If unavoidable, place it on a 16-radius squircle plate first.
